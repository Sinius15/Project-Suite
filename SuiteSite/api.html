These are the api functions:

[game] => The game you want to have. Avalable games are "suite", "chess", "maze".
  • "portal.php?game=[game]&req=versionList" returns a list of all downloadable versions.
  • "portal.php?game=[game]&req=latestVersion" returns the latest avalable version.
  • "portal.php?game=[game]&req=login&name='username'&pass='password'" returns true or flase, depending if the username/password combination is correct.
  • "portal.php?game=[game]&req=existsPlayer&name='username'" returns true or fase, depending if the user exists
  • "portal.php?game=[game]&req=existsEmail&mail='email address'" returns true or false, depending if the email address is used by a player.
